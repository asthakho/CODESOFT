01
Problem Statement

What are we solving?

Businesses invest heavily in advertising — TV, Radio, and Newspaper — but how much does each dollar spent actually generate in sales? This project builds a Simple Linear Regression model to answer that question.

We use the classic ISLR Advertising Dataset (200 records) and focus on the relationship between TV advertising spend and product sales, since TV shows the highest correlation.

📺
TV Advertising
Primary predictor. Correlation of 0.78 with Sales — the strongest among all channels.
📻
Radio
Moderate correlation of 0.58. Less impactful but still significant for multi-linear models.
📰
Newspaper
Weak correlation of 0.23 with Sales. Excluded in this simple model.
💰
Sales (Target)
Units sold (in thousands). What we are predicting using the linear model.
02
Data & Cleaning

Understanding the data

The dataset contains 200 rows and 4 columns: TV, Radio, Newspaper (advertising budgets in $000s), and Sales (in 1000 units). There are no null values and no significant outliers.

# Import libraries import numpy as np import pandas as pd import matplotlib.pyplot as plt import seaborn as sns # Load dataset advertising = pd.read_csv("advertising.csv") advertising.head() # Shape: (200, 4) | No null values found # Correlation heatmap confirms TV is strongest predictor sns.heatmap(advertising.corr(), cmap="YlGnBu", annot=True)
03
Model Building

Building the regression model

We use Ordinary Least Squares (OLS) via statsmodels to fit the line. The data is split 70/30 into training and test sets.

01
Feature Selection
TV is selected as the sole predictor (X) and Sales as the target (y), based on pairplot and correlation analysis.
02
Train / Test Split
70% of data (140 rows) used for training, 30% (60 rows) held out for evaluation. random_state=100 for reproducibility.
03
OLS Fit with Intercept
sm.add_constant() adds the intercept term. lr = sm.OLS(y_train, X_train_sm).fit() trains the model.
04
Residual Analysis
Error terms follow a normal distribution (mean ≈ 0), confirming that key linear regression assumptions are met.
05
Test Set Prediction
X_test_sm = sm.add_constant(X_test) then y_pred = lr.predict(X_test_sm). Final RMSE and R² are computed.
from sklearn.model_selection import train_test_split import statsmodels.api as sm X = advertising['TV'] y = advertising['Sales'] X_train, X_test, y_train, y_test = train_test_split( X, y, train_size=0.7, test_size=0.3, random_state=100 ) X_train_sm = sm.add_constant(X_train) lr = sm.OLS(y_train, X_train_sm).fit() print(lr.summary()) # Intercept: 6.948 | TV Coefficient: 0.054
04
Results

Model performance

The model performs well on both train and test sets, confirming it generalises without overfitting.

0.816
R² Score (Train)
81.6% of variance in Sales is explained by TV spend alone.
0.794
R² Score (Test)
Strong generalisation — model holds on unseen data.
1.73
RMSE (Test)
Root Mean Squared Error in units of Sales (1000s).

The p-value for the TV coefficient is essentially zero, confirming the relationship is statistically significant and not by chance. The F-statistic also confirms overall model significance.
